Steps to Contribute

Hi friend, it's so great that you want to contribute! If you are familiar with Github and version control, or want to give it a try (and we will help you if you run into trouble!) follow the instructions at the repository. If not, please read the list of requirements, and fill in the following form (full page here), and we will contact you to get your image file. Please read these requirements carefully, as missing one will delay adding your contribution to the gallery.

Image Requirements

1. There is technically no limit to the size and resolution of your image, however images with the largest dimension size 1024 tend to come out better. If your image is so large that it goes over the time limit of our robot artists, then we will ask you for a smaller image to add to the gallery.

2. All images must be jpg, with extension "jpg"

3. Image file names must be all lowercase, with numbers and special characters `-` and `_` allowed

We strongly recommend you contribute via the automated approach on Github (and learn about version control!) for a much faster addition. Otherwise, fill out the full page form here, and thank you for your contribution!
